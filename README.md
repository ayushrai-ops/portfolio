# Portfolio

My personal portfolio website. Built with React + Vite for the frontend and a simple Express server for the contact form.

## Stack

- React, Vite, Tailwind CSS v4
- Three.js + React Three Fiber for the hero section
- Framer Motion for animations
- Express backend for contact form

## Getting Started

```bash
# frontend
cd client
npm install
npm run dev

# backend (separate terminal)
cd server
npm install
node index.js
```

Frontend runs on `http://localhost:5173`, backend on `http://localhost:5000`.

## Deployment

Frontend can be deployed on Vercel/Netlify. The server needs a separate host like Render or Railway — just update the API URL in the client accordingly.
